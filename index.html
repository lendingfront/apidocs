LendingFront API

The LendingFront API is a simple REST based API that allows you to submit application, data source, decision, underwriting and loan requests to LendingFront’s platform.

Our platform collects Social, Cash Flow and Credit Bureau data electronically. It is important that the business owner agrees to the terms and conditions before requests are submitted.

Authentication

LendingFront's APIs use token based authentication. You can get a key in the Settings page of the 
LendingPortal. The setting page is only available to Administrator users.

During the initial pilot, LendingFront will manually provide you with a key.

API calls must be sent via https (SSL)

Testing Host

During the implementation phase, LendingFront will provide the test and production URLs.
